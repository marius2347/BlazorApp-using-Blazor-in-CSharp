A modern, interactive web application built with **ASP.NET Core Blazor**. This project demonstrates the power of building client-side web UIs using **C#** and **.NET** instead of JavaScript, following the core concepts of component-based architecture.

## üöÄ Features

* **Interactive Counter:** A dynamic component that demonstrates **state management** and **event handling** in Blazor. Users can interact with the UI to update the count in real-time without a full page reload.
* **Component-Based Architecture:** The application is built using reusable Razor components (`.razor`), showcasing how UI logic and layout can be encapsulated.
* **Single Page Application (SPA) Feel:** Provides a smooth user experience with fast client-side navigation and updates.
* **Responsive Layout:** Uses the standard Blazor layout structure with a navigation menu and main content area.

## üìö Key Concepts Learned

This project implements fundamental Blazor concepts:
* **Razor Components:** Defining UI elements and logic in a single file.
* **Data Binding:** Linking UI elements to C# variables.
* **Event Handling:** Responding to user actions (like button clicks) using C# methods (e.g., `@onclick`).
* **Dependency Injection:** Using built-in services within components.

## üì¶ Getting Started

To run this application locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/marius2347/BlazorApp-using-Blazor-in-CSharp.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd BlazorApp-using-Blazor-in-CSharp
    ```
3.  **Run the application:**
    ```bash
    dotnet run
    ```
4.  Open your browser and navigate to the local URL shown in the terminal (usually `https://localhost:7000` or similar).
